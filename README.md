# 萝卜还是纸巾？——自拍视频小游戏（Vue + Vite）

把“萝卜/纸巾选择题 + 选对就夸你「真棒」”做成一个**真人出镜**的小游戏：  
用摄像头画面当背景，你会被实时“装扮成小猫”（猫耳/胡须叠加），系统随机喊物品，你用**把脸移到左/右选区**来做选择；选对会有“好吃的”从天上掉下来，张嘴还能“吃到”。支持**录制并下载** webm。

灵感来源文章：`https://news.china.com/socialgd/10000169/20251224/49104115.html`

## 运行

```bash
npm install
npm run dev
```

打开 `http://localhost:5173/`，允许摄像头权限即可。

### 手机通过 IP 访问看不到视频/不弹权限？

多数手机浏览器在 **HTTP + IP**（例如 `http://172.x.x.x:5173`）下会直接禁用摄像头。现在项目已开启 **HTTPS dev**：

- 用手机打开：`https://你的IP:5173/`
- 首次会提示证书不安全：选择“继续访问/仍然访问”
- 建议用系统浏览器（Chrome/Edge）打开，部分 App 内置浏览器会拦截摄像头权限弹窗

## 操作说明

- **开始考试**：进入出题状态
- **选答案**：把脸移到画面左/右下角选区停留（可调速度）
- **奖励**：答对掉落🐟/🍗，张嘴可“吃到”
- **录制**：勾选“录制”后可开始/停止并自动下载视频
